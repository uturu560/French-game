FAQ

What is this game?

A free French–English vocabulary matching game. Flip cards to find pairs (e.g. chien and dog) before time runs out.

How do I play?

Choose a theme (Animals, Colors, Numbers, Family, Verbs), pick a level, memorize the cards when they’re shown, then click two at a time to match. Match all pairs before the timer ends to win. Get 2+ stars to unlock the next level.

Is it free?

Yes. The game is free to play. You can support it via the Support the game button in the header on this page.

Who is it for?

Anyone learning or brushing up French or English — students, teachers, and casual learners. The interface can be switched to English or Français.

How can I support the game?

Use the Support the game button above. You’ll be redirected to a new tab. Thank you!

About

This is a free French–English vocabulary matching game. You see cards with French words on some and English words on others. Your goal is to find matching pairs before the timer runs out.

How to play (in detail)

  • Choose a set — Pick a theme (Animals, Colors, Numbers, Family, Verbs) from the dropdown.
  • Pick a level — Levels 1–10; each level uses 6 pairs. Unlock the next level by finishing the current one with at least 2 stars.
  • Memorize — Cards are shown face-up briefly, then flipped. Remember where each word is.
  • Match — Click two cards to flip them. If they match (French–English pair), they stay face-up. Match all pairs before time runs out to win.
  • Score — You earn points for matches and a time bonus for finishing quickly.

Anyone learning or brushing up French or English can use the game. Switch the interface language with the toggle on the game page. If you enjoy it, you can support the game via the button in the header above.
